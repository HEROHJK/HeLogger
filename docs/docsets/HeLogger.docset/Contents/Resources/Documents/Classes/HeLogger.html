  • formatString

    로그 방식 포맷팅. 지원 종류는 다음과 같음.

    • {$level}: 로그 레벨
    • {$type}: 로그 타입
    • {$time}: 현재 시각
    • {$function}: 호출된 이전 함수
    • {$file}: 호출한 함수의 파일 명
    • {$line}: 호출한 함수 파일의 라인
    • {$message}: 로깅 내용

    public var formatString: String
